## 🚀 How to Open & Use the Dashboards

### Option 1 — Tableau Desktop (Full Access)

1. **Clone this repository**
   ```bash
   git clone https://github.com/your-username/tableau-projects.git
   cd tableau-projects
   ```

2. **Open a dashboard**
   - Launch **Tableau Desktop**
   - Go to `File → Open`
   - Navigate to the project folder and open the `.twbx` file

3. **Connect to data** *(if prompted)*
   - `.twbx` files include packaged data — no extra setup needed
   - For `.twb` files, point Tableau to the matching data file inside `/data/`

4. **Explore & interact**
   - Use filters, dropdowns, and date selectors to explore the data
   - Hover over charts for detailed tooltips

## 📌 Project Highlights

### 💰 Revenue Overview Dashboard
- Visualizes monthly, quarterly, and annual revenue trends
- Includes YoY (Year-over-Year) comparison charts
- Filterable by region, product line, and sales channel

### 📉 Expense Tracker
- Breaks down operational expenses across departments
- Highlights cost anomalies and overspending categories
- Drill-down capability from summary to transaction level

### 📈 KPI Monitor
- Single-view scorecard of critical business KPIs
- Color-coded performance indicators (on track / at risk / off track)
- Supports executive-level reporting

### 📋 Quarterly Performance Report
- Side-by-side quarter comparison with variance highlights
- Automated target vs. actual tracking
- Exportable for stakeholder presentations

### 🧮 Budget vs. Actual Analysis
- Tracks spending relative to approved budgets
- Variance waterfall charts for quick insights
- Filterable by department, cost center, and period

### 📊 Profitability Analysis
- Gross and net margin analysis by product/service line
- Trend lines for profit over time
- Segment-level profitability breakdown
